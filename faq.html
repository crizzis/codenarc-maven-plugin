How do I use a different version of CodeNarc?

I will try to do some housekeeping around the plugin from time to time, bumping up the CodeNarc version. In the meantime, if you desperately need to use a different version of CodeNarc (e.g. an urgent bug fix), you may try to update the dependency yourself using the <dependencies> section of the <plugin> configuration.

[top]

Can I use codenarc.properties?

Sure, knock yourself out. Just put them in the resource root directory of your project.
